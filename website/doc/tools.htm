Test with input

A general purpose function for testing

Get test results

REQUEST

GET

https://signpuddle.net/v3/tools/test{?text}
Parameters
text string S10000

Input for testing

RESPONSE

Headers
Content-Type text/plain
200 text/plain
BODY SCHEMA 
                        test output

Part definition

A part of the section definition

Get part define

REQUEST

GET

https://signpuddle.net/v3/tools/define/{section}/{part}
Parameters
section string swu

The name of section
part string symbol

The part of the definition

RESPONSE

Headers
Content-Type text/plain
200 text/plain
BODY SCHEMA 
                        [
  "individual symbol",
  "S10000",
  "[\\x{40000}-\\x{4F428}]"
]
Parse text

A function to analyze text and parse it into individual components

Get parse results

REQUEST

GET

https://signpuddle.net/v3/tools/parse{?text,utf}
Parameters
text string S10000

The text to parse
utf string 32

The UTF number of 8, 16, 32 or ‘x’

RESPONSE

Headers
Content-Type text/plain
200 text/plain
BODY SCHEMA 
                        parse results of text
Encode text

A function to encode SignWriting in Unicode (SWU) as UTF-16

Get encoded query string

REQUEST

GET

https://signpuddle.net/v3/tools/encode{?text,slash}
Parameters
text string \x{1D800}

The text to encode
slash number 1

The number of slashes for escaping

RESPONSE

Headers
Content-Type text/plain
200 text/plain
BODY SCHEMA 
                        \uD836\uDC00
