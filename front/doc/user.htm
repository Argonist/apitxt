SignPuddle 3 collections are organized by country and language codes

Country code

The country codes are from ISO-3166. Each country is coded with two uppercase letters.

Language code

The language codes are from ISO-639-1 for spoken languages and ISO-639-3 for sign languages. Each spoken language is coded with two lowercase letters. Each sign language is coded with three lowercase letters.
